Cara Menggunakan Tipe Data di Javascript

1. Tipe Data Integer/Number (bilangan bulat)

2. Tipe Data String (text)

3. Tipe Data Boolean (tipe data dengan nilai true(1) & False(0))

4. Tipe Data Array (sekumpulan nilai yang urutan pertama dihitung dari indeks ke-0)

5. Tipe Data Object (sekumpulan nilai yang memiliki parent)
